## Hot to start?

Create a `.env` file at the root of the project, please, refer to the `env.example` to add the values with user specific.


## Installation

```bash
$ npm install

$ npm run migration:run
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
